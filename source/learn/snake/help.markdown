贪吃蛇操作说明
=========================== 
---
date: 2012-12-13
---

## 操作说明
		1. 在单人模式下，使用上下左右控制贪吃蛇；
		2. 在双人模式下，黑蛇使用上下左右控制，红蛇使用WASD控制；
		3. 绿色块为基本食物，能增长贪吃蛇；
		4. 非绿色食物能触发特殊效果；
		5. 在V3.0版本后使用P键暂停游戏。

## 温馨提示
		1. 初学JavaScript，欢迎指出错误；
		2. 没有进行bug测试，特别是双人模式，欢迎指出bug；
		3. 系统还未完善，还在不断更新中；
		4. 有好的建议欢迎和我联系。
